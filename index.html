About Me

Hello and welcome! My name is Ross Dimchev, and I am an aspiring analytics professional. I am originally from Bulgaria and came to the U.S. in 2013. After finishing high school at Lawrenceville Prep, I earned a scholarship to the University of Richmond. I graduated with honors in 2020 and received a B.S. in Business Administration with concentrations in Finance and Economics.

Following graduation, I started working in the mortgage industry and eventually became a Loan Officer, licensed in 5 states. As grateful as I am for all the skills and experience, I gained from working in financial services, in early 2023, I decided to take my career in a different direction. I discovered the world of data and realized the immense potential of this ever-expanding domain. Although I had years of experience working with Excel and exposure to data analytics through various undergraduate courses, I felt like I needed a refresher. I found the Google Data Analytics certificate, which became my first steppingstone into data analytics. Ever since the certificate's final project, I have been hooked on taking new courses and teaching myself the necessary skills.

My goal is to find a career in data/business analytics. More importantly, I aim to learn, improve, and contribute.

SQL Projects

  • COVID-19 SQL Exploration

I am using publicly available COVID-19 data from Our World in Data. I downloaded the information from the start of the pandemic until early May, 2023 as a CSV file. I used Excel to divide the data into two tables so I can practice joins. I have used SQL Server Management Studio to write the queries in T-SQL, after importing the Excel files to MS SQL Server.

  • SQL Data Cleaning

I am using data set from GitHub on the Nashville, TN housing market. The data was imported into SQL Server Management Studio as an Excel file and various functions were used to clean up the data and make it more usable.

Tableau Projects

  • COVID-19 Dashboard

I am using publicly available COVID-19 data from Our World in Data. This is the same data set used for my first SQL project. In a work setting, I would have been able to connect Tableau to the SQL server and query directly. Since I don't have the paid version of Tableau I had to run each query I wanted to visualize and copied its contents to Excel files, which I imported to Tableau Public.

  • Asheville, NC Airbnb Dashboard

I am using data from Inside Airbnb. I decided to work on the Asheville market since I love visiting the area and was curious about investing there. The data was downloaded as 2 separate CSV files that were converted to Excel and then joined in Tableau based on the listing ID.

Power BI Projects

  • Data Professional Survey Dashboard

I am using data from GitHub. The data was imported into Power BI from an Excel file and cleaned through Power Query. The survey data set contains useful information from 2022 on how data professionals feel about various aspects of their career.
